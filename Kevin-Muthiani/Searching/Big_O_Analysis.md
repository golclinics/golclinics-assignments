# 1. Sum Pair
## 1.1 Linear Search 
### Arrays

**Time: ** O(n^2)  
**Space: ** O(1)  
`n` - size of nums list  

### Hash Maps

**Time: ** O(n)  
**Space: ** O(n) 
`n` - size of nums list  

## 1.2 Binary Search - Recursive

**Time: ** O(log(n))  
**Space: ** O(log(n))  
`n` - size of nums list  

## 1.3 Binary Search - Iterative

**Time: ** O(log(n))  
**Space: ** O(1)  
`n` - size of nums list  

# 2. Student Grades
## 2.1 Linear Search

**Time: ** O(m + mn)  
**Space: ** O(m + mn)  
`n` - size of students list  
`m` = size of grades list  

## 2.2 Binary Search - Recursive

**Time: ** O(m + n + mlog(n))  
**Space: ** O(m + n + mlog(n))  
`n` - size of students list  
`m` = size of grades list  

## 2.3 Binary Search - Iterative

**Time: ** O(m + n + mlog(n))  
**Space: ** O(m + n)  
`n` - size of students list  
`m` = size of grades list
